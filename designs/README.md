These files are editable SVG mockups sized for Android HD (1080×2340).

How to import into Figma:
1. Open Figma and create a new file.
2. Drag and drop any `.svg` file from this folder onto the canvas.
3. Ungroup or edit vector elements as needed.

To export PNGs from Figma:
- Select the frame or imported SVG, then in the right-hand panel choose Export > PNG and set scale (1x, 2x, etc.).
